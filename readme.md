# Introduction
it's a simple tool for collecting East Asia weather data  

# How to use  
## to download  
1. install python v3 or later  
2. python.exe eawdc.py  
3. you can find data in the folder named "data"  

## to modify the time configs
modify configloader  

## to modify urls
modify urlgenerator
